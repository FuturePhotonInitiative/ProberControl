Installation¶

  1. Install Python 2.7.11
    • Python Download Link to Python 2.7.11 for Windows machines
    • When installing Python, be sure to choose to Add Python to path option.
  1. Install National Instruments driver, so you can interface with PyVisa
    • Driver Download Link
  2. Install PyGTK
    • PyGTK Download Link
  3. If you don’t have Visual C++ from Microsoft, you’ll need to download that as well.
    • Link
  4. You’ll likely have to restart your machine at this point.
  5. Run install.bat
    • Located in /ProberControl/setup
  6. Congratulations! You should be able to move over to /ProberControl/prober/ and run ProberControl.py which will launch the GUI for ProberControl
